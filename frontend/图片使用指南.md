如果暂时没有图片文件，系统会使用占位图片。

## 图片规格建议

### 功能卡片图片
- **尺寸**: 800 x 450 像素 (16:9 比例)
- **格式**: JPG, PNG, WEBP
- **大小**: < 500KB
- **内容**: 清晰的产品或场景图

### 作品展示图片
- **尺寸**: 600 x 800 像素 (3:4 比例)
- **格式**: JPG, PNG, WEBP
- **大小**: < 500KB  
- **内容**: 高质量的作品展示

## 快速开始

### 步骤1: 准备图片
将您的图片重命名为上述文件名

### 步骤2: 复制到项目
```bash
# 复制功能卡片图片
cp /path/to/your/image1.jpg frontend/public/images/features/smart-edit.jpg
cp /path/to/your/image2.jpg frontend/public/images/features/scene-change.jpg
# ... 继续复制其他图片

# 复制作品图片
cp /path/to/your/artwork1.jpg frontend/public/images/artworks/artwork-1.jpg
# ... 继续复制其他图片
```

### 步骤3: 刷新浏览器
图片会自动显示，如果没有图片文件，会显示占位图。

## 图片优化建议

### 压缩图片
使用在线工具压缩图片：
- TinyPNG: https://tinypng.com/
- Squoosh: https://squoosh.app/

### 格式转换
建议使用 WebP 格式，文件更小，质量更好：
```bash
# 使用命令行转换 (需要安装 webp 工具)
cwebp input.jpg -q 80 -o output.webp
```

## 注意事项

1. **版权**: 确保您有权使用这些图片
2. **尺寸**: 保持统一的宽高比
3. **命名**: 使用英文命名，避免空格和特殊字符
4. **大小**: 单张图片不要超过1MB
5. **备份**: 保留原图备份

## 问题排查

### 图片不显示？
1. 检查文件名是否正确
2. 检查文件路径是否正确
3. 刷新浏览器缓存 (Ctrl+Shift+R / Cmd+Shift+R)
4. 检查浏览器控制台是否有错误

### 图片显示模糊？
1. 使用更高分辨率的原图
2. 避免过度压缩
3. 使用2x分辨率的图片以支持高清屏幕
